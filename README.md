<h1>HAWAII HWI</h1>

<p>Hawaii is a Quark based cryptocurrency with PoW, PoS, and long term support (LTS). Hawaiicoin uses the rock solid 9-hash algorithm Quark for Proof of Work and the Novacoin for Proof of Stake.</p>
